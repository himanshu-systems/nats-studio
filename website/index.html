Everything, in one window

The tool NATS was missing

A real desktop client — not a CLI wrapper, not a web tab. Fast, native, and built for how you actually work with NATS.

Live Tail

Subscribe to any subject and watch messages stream in, with per-subscription filtering and instant search.

Publish anything

Compose JSON, Protobuf, MessagePack, Text, Hex, or Base64 — with templates, variables, and burst publishing.

JetStream

Create and inspect streams and consumers, browse stored messages, and manage retention — no CLI required.

KV & Object Store

Read and write Key-Value buckets, and stream large objects to and from disk with live progress.

Request / Reply & Services

Fire request/reply calls and discover NATS micro-services with per-endpoint stats and schemas.

Secure by default

TLS & mTLS profiles, credentials stored in the OS keychain, and zero telemetry. Your data stays yours.
